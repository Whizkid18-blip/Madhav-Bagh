MADHAV BAGH — Royal Heritage Stay  ·  website
=============================================

A complete website. Here is how to put it online for FREE with a
permanent link you can send to anyone — no coding needed.

------------------------------------------------------------
PUT IT ONLINE WITH GITHUB PAGES  (free, permanent link)
------------------------------------------------------------

1. Go to  https://github.com  and sign in.

2. Click the  "+"  (top-right)  ->  "New repository".
       Repository name:   madhav-bagh
       Visibility:        Public
   Click  "Create repository".

3. On the new repo page, click the blue link
   "uploading an existing file".
   Open this folder, select EVERYTHING inside it —
       index.html, css, js, assets, README.md, .nojekyll
   — and drag it all onto the page together.
   (Give the "assets" folder a moment; it holds the photos.)
   Then scroll down and click  "Commit changes".

4. Turn on the free website:
       Repo  "Settings"  ->  "Pages"  (left-hand menu).
       Under "Build and deployment", Branch = "main", folder = "/ (root)".
       Click  Save.
   Wait about a minute, refresh the page. GitHub shows a green
   link like:
       https://YOURNAME.github.io/madhav-bagh/
   THAT is your live website. Send that link to anyone.

------------------------------------------------------------
NOTES
------------------------------------------------------------
- The 3D effect and the fonts load from the internet, so the page
  needs a connection (always the case for any website).
- Every photo is a real photo of Madhav Bagh — and yes, Stella and
  Zephyr are both in there. Everything is already inside "assets".
- Best experienced by scrolling slowly, top to bottom. It is meant
  to feel like walking through the house.

Prefer Replit instead? Create an "HTML, CSS, JS" Repl, delete its
three starter files, drag everything from this folder in, press
Run, then Deploy -> Static for a link.

Made with love. Scroll slowly.
